Quad-Tree Program

Uses a quad-tree algorithm to select all the nodes within the radius. Move your mouse around and explore.
Keyboard commands below.

Keyboard Commands:
[space]: toggle quad tree lines (on/off)

[a | t]: add [100 | 1000] random points
Numbers 1-4: change radius size

p: increase size of canvas
o: decrease size of canvas
r: remove all points

Mouse Click: add 30 points close to mouse
